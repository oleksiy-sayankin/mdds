# MDDS Python Worker Runtime

Python Worker Runtime for the MDDS job orchestration platform.

This package provides the runtime infrastructure for executing MDDS worker jobs,
including job message consumption, validation, status publishing, cancellation
handling, supervised execution, artifact preparation and output upload.
